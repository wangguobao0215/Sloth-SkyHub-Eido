## Features

- **Full Customer Journey Coverage**: 6 lifecycle stages (Handoff → Onboarding → Adoption → Value Realization → Expansion → Renewal) with 17 child Skills mounted on journey nodes
- **Three-Tier Segmentation**: Strategic (AI as strategist), Growth (AI as co-pilot), Scale (AI as lead actor) — each Skill auto-adapts service depth
- **Health Score Engine**: 7-dimension rule-based scoring with stage-dynamic weight adjustment
- **Playbook Engine**: 10 core standardized process playbooks with trigger → steps → branches → escalation → metrics
- **Value Realization Core**: Shifts narrative from "what we did" to "what you gained" — automated ROI calculation, industry benchmarking, value narrative generation
- **Local-First Deployment**: Built on Obsidian/Markdown knowledge base, data sovereignty, offline-capable, Xinchuang-friendly
- **Graceful Degradation**: Every Skill includes G1/G2/G3 fallback strategies — "lossy degradation over total failure"
- **China Market Native**: IM-first communication, relationship-driven renewals, WeChat Work/DingTalk integration, SOE compliance

## Quick Start

1. **Install**: Copy `Sloth-SkyHub-Eido` folder to `~/.qoderwork/skills/`
2. **Initialize KB**: Create `SkyHub-KB` Vault in Obsidian following [obsidian-vault.md](references/obsidian-vault.md)
3. **Import Data**: Import existing customer data into `01-Customers/`
4. **Activate**: Send "SkyHub start" in QoderWork or ask any customer-related question
