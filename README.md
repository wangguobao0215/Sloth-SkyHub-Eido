## 品名释义

> **深信**——「深」取自树懒老K品牌哲学「慢一点，深一度」，「信」意为信赖、信任。客户成功的根基是深层信赖关系，不是表面的续费数字。深信，以深度信赖构建客户终身价值。

## 功能概览

- **客户旅程全覆盖**：6个生命周期阶段（销售交接 → 启用上线 → 深度采纳 → 价值兑现 → 扩展增购 → 续约），17个子Skill按旅程节点挂载
- **三层客户分级**：Strategic（AI为幕后军师）、Growth（AI为副驾驶）、Scale（AI为主角），每个Skill自动适配服务深度
- **健康评分引擎**：7维度规则化评分（usage_depth、activity_trend、commercial_health、support_tickets、stakeholder_engagement、value_realization、sentiment），权重按旅程阶段动态调整
- **Playbook引擎**：10个核心标准化流程剧本，触发条件 → 步骤序列 → 分支判断 → 升级机制 → 度量闭环
- **价值实现为内核**：从"我们做了什么"转向"你获得了什么"，ROI量化、行业基准对比、价值叙事自动生成
- **本地优先部署**：基于Obsidian/Markdown知识库，数据主权可控，支持离线运行，信创友好
- **优雅降级保障**：每个Skill内置G1/G2/G3三级降级策略，"有损降级优于完全失败"
- **中国市场深度适配**：IM-first沟通范式、关系型续约、企微/钉钉话术、国企合规

## 快速开始

1. **安装技能**：将 `Sloth-SkyHub-Eido` 文件夹复制到 `~/.qoderwork/skills/` 目录
2. **初始化知识库**：在 Obsidian 中创建 `SkyHub-KB` Vault，按 [obsidian-vault.md](references/obsidian-vault.md) 结构初始化
3. **导入客户数据**：将现有客户信息导入 `01-Customers/` 目录
4. **激活使用**：在 QoderWork 中发送 "SkyHub 启动" 或直接提问客户相关问题
