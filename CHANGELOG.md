## [1.0.0] - 2026-04-23

### Added

- Initial release of Sloth-SkyHub-Eido v1.0.0
- SKILL.md with full intent routing and 17 child Skill module definitions
- Customer journey-driven architecture (6 stages: Handoff -> Onboarding -> Adoption -> Value Realization -> Expansion -> Renewal)
- Three-tier customer segmentation (Strategic / Growth / Scale) with differentiated AI behavior
- Parent agent "Tianshu Hub" with 3-level context management (L1 Hot / L2 Warm / L3 Cold)
- Health score model with 7 dimensions and stage-dynamic weights
- Playbook engine with 10 core Playbooks
- L1-L5 automation level governance framework
- G1-G3 graceful degradation protocol for all Skills
- Security & compliance module (S1-S4 data classification, SOE/Xinchuang support)
- 4-phase implementation roadmap (M1-M4 Foundation -> M4-M8 Process -> M8-M14 Data-Driven -> M14+ Scale)
- 10-risk mitigation matrix (R1-R10)
- 12 reference files covering journey, tiers, skills, health model, playbooks, metrics, security, Obsidian vault, MCP integration, roadmap, risks, and cross-skill integration
- README.md (Chinese) and README_EN.md (English) with brand identity
- Full Sloth-Eido family convention compliance
